制造行业的ERP系统

该系统是面向各生产制造行业而开发的一款合理有效的企业资源规划系统。核心是进销存，以及销售，财务等管理。系统是分各个子模块开发的，主要模块有采购、销售、仓库、财务、报表、材料等模块。

这里是我主要负责的模块内容代码：权限模块、仓库模块
